PILOTS - RESTful Service API Documentation

    General Description

    This is a general-purpose platform for creators to help bring creative projects to life with the help of consumers who decide which project they are willing to contribute credits to, and as a result help creators to reach the goal they set for themselves, helping creators to publish their projects. This model tracks its roots to subscription model of arts patronage, where artists would go directly to their audiences to fund their work

  • Producer

    • Schema

    {
    id: mongoose.Schema.Types.ObjectId,
    user_name: String,
    password: String,
    full_name: String,
    phone: String,
    email: String,
    projects: [{ type: mongoose.Schema.Types.ObjectId, ref: "ProjectSchema" }]
}

    /producer/

    Request Method: GET

    Response: all existing producers in the system.

    URL: https://pilots.herokuapp.com/producer 

                                JSON EXAMPLE

    /producer/createProfile/

    Request Method: POST

    Parameters: user_name, password, full_name, phone, email

    Response:: success/fail regarding user creation.

    URL: https://pilots.herokuapp.com/producer/createProfile 

                                JSON EXAMPLE

    /producer/getProfile/:id

    Request Method: GET

    Response:: all matching id's profile details.

    URL: https://pilots.herokuapp.com/producer/getProfile/:id 

                                JSON EXAMPLE

    Parameters: user_name, password, full_name, phone, email

  • Project

    • Schema

    {
    id: mongoose.Schema.Types.ObjectId,
    category: String,
    owner: { type: mongoose.Schema.Types.ObjectId, ref: "ProducerSchema" },
    title: String,
    goal: Number,
    positive_voters: [
        { type: mongoose.Schema.Types.ObjectId, ref: "ConsumerSchema" }
    ],
    negative_voters: [
        { type: mongoose.Schema.Types.ObjectId, ref: "ConsumerSchema" }
    ],
    goal_status: Number,
    open_timestamp: Date,
    description: String,
    deadline: Date,
    subscribers: [
        { type: mongoose.Schema.Types.ObjectId, ref: "ConsumerSchema" }
    ]
}
